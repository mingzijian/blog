# 什么是打印机

打印机（Printer）是计算机的[输出设备](https://baike.baidu.com/item/输出设备/10823333)之一，用于将计算机处理结果打印在相关介质上。

# 常见的打印机

## 打印机种类

### 按工作原理分

| 种类       | 组成                 | 原理                                                         | 应用场景                                                     |
| ---------- | -------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 针式打印机 | 针头、色带           | 打印时针头按压色带，将文字印到纸上                           | 针式打印机技术简单，输出颜色单一，所以主要用于发票、物流回执等低覆盖率的文字输出。 |
| 喷墨打印机 | 喷头、墨盒           | 打印时喷头将墨水喷射到纸上形成文字或图像。<br>微压电：喷头加电发生形变把墨水挤压出去<br>热发泡：碰头加热把墨水变成水蒸气喷出去<br> |                                                              |
| 激光打印机 | 鼓粉、硒鼓、加热组件 | 由计算机传来的二进制数据信息，通过视频控制器转换成视频信号，再由视频接口/控制系统把视频信号转换为激光驱动信号，然后由激光扫描系统产生载有字符信息的激光束，最后是由电子照相系统使激光束成像并转印到纸上。 |                                                              |
