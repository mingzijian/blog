### Eureka 对比 zookeeper

| 对比项       | Eureka                                                       | Zookeeper                                        |
| ------------ | ------------------------------------------------------------ | ------------------------------------------------ |
| CAP 设计原则 | AP                                                           | CP                                               |
| 优点         | 服务高可用                                                   | 数据强一致                                       |
| 缺点         | 服务节点间的数据可能不一致；Client-Server 间的数据可能不一致； | 网络分区会影响 Leader 选举，超过阈值后集群不可用 |
| 适用场景     | 云机房集群，跨越多机房部署；对注册中心服务可用性要求较高     | 单机房集群，对数据一致性要求较高                 |
