JMH
-------
JMH是一个Java工具，用于构建、运行和分析用Java和其他针对JVM的语言编写的基准测试。
支持的时间单位：nano/micro/milli/macro

JMH只适合细粒度的方法测试，并不适用于系统之间的链路测试！
