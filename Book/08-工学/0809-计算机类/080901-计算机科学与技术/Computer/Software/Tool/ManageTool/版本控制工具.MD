### SVN

#### 简介

SVN是subversion的缩写，是一个开放源代码的版本控制系统，通过采用分支管理系统的高效管理，简而言之就是用于多个人共同开发同一个项目，实现共享资源，实现最终集中式的管理。

#### 参考
 - http://subversion.apache.org/


### Git
Git 是一个开源的分布式版本控制系统，用于敏捷高效地处理任何或小或大的项目。

### GIT 安装
查看当前GIT安装版本
```shell
git --version
```
[GIT 下载](https://git-scm.com/download/)

#### 命令安装
- CentOS
```shell
yum install -y git
```
- Debian/Ubuntu/Deepin
```shell
apt-get install git
```
#### 源码安装
[GIT 镜像](https://mirrors.edge.kernel.org/pub/software/scm/git/)

```shell
# 下载
> wget https://mirrors.edge.kernel.org/pub/software/scm/git/git-2.29.2.tar.gz
# 解压
> tar -zxvf git-2.29.2.tar.gz
> cd git-2.29.2
# 编译前先安装编译依赖
> yum install -y curl-devel expat-devel gettext-devel openssl-devel zlib-devel gcc perl-ExtUtils-MakeMaker
# 配置安装路径
> ./configure --prefix=/usr/local/git
# 编译安装
> make && make install

# 配置环境变量
> vim /etc/profile
# GIT_HOME
GIT_HOME=/usr/local/git
export PATH=$PATH:$GIT_HOME/bin

# 使环境变量配置生效
> source /etc/profile

> git --version

```

### GIT  使用

#### 代码量统计

```shell
# 2023年6月半山代码统计
git log --author=半山 --pretty=tformat: --numstat --after="2023-06-01 00:00:00" --before="2023-07-01 00:00:00" | awk '{ add += $1 - $2 } END { printf "总行数: %s\n", add }'

git log --author=半山 --pretty=tformat: --numstat --after="2023-06-01 00:00:00" --before="2023-07-01 00:00:00" | awk '{ add += $1 } END { printf "新增行数: %s\n", add }'

git log --author=半山 --pretty=tformat: --numstat --after="2023-06-01 00:00:00" --before="2023-07-01 00:00:00" | awk '{ add += $1 } END { printf "删除行数: %s\n", add }'

```
