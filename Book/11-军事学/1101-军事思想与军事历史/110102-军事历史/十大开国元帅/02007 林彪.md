          1937 : 被阎锡山部哨兵误伤，伤及肺和脊椎骨，留下严重后遗症，同年冬赴苏联就医
          1942 : 从苏联回国
          1945 : 在重庆与周恩来一同参与国共谈判；当选为中共中央委员、中央军委委员
          1948 : 指挥辽沈战役，解放全东北；指挥平津战役，攻克天津，和平解放北平（今北京）
          1949 : 任第四野战军司令员等职，先后指挥了湘赣、衡宝、广东、广西等战役，解放中南广大地区
          1950 : 旧伤引发的病症加重，多次赴苏联就医
          1954 : 任国务院副总理和国防委员会副主席
          1955 : 被授予元帅军衔
          1958 : 增选为中共中央政治局常委、中共中央副主席
          1959 : 庐山会议后，林彪兼任国防部长并主持军委日常工作
          1969 : 在中共九大上，作为毛泽东接班人被写进党章
          1970 : 在庐山召开九届二中全会，提议修宪恢复国家主席，明确国家副主席人选，未通过
          1971 : 在蒙古温都尔汗地区坠机身亡
          1973 : 被开除党籍
          1981 : 被最高人民法院特别法庭确认为反革命集团案主犯
