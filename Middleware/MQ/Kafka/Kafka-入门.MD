Kafka 入门
----------------
Apache Kafka 是 一个分布式流处理平台。

### 特性
1. 发布和订阅流式记录（MQ）
2. 储存流式记录，并且有较好的容错性
3. 在流式记录产生时进行处理


### 应用场景
1. 构造实时流数据管道（当MQ用）
2. 构建实时流式应用程序，对这些流数据进行转换或者影响。（流处理）
3. 充当中间数据的存储系统

### 核心API
1. Producer API
	允许应用程序发布一串流式数据到一个或者多个topic
	
2. Consumer API
	允许应用程序订阅一个或者多个topic，并且对订阅到的流式数据进行处理
	
3. Streams API
	允许一个应用程序作为一个流处理器，消费一个或者多个topic产生的输入流，然后生产一个输出流到一个或者多个topic中去，在输入输出流中进行有效的转换
	
4. Connector API
	允许构建并运行可重用的生产者或者消费者，将topics连接到已存在的应用程序或者数据系统
	
5. Admin API
    管理和检查 topics, brokers, acls和其他Kafka对象
    
    
### Kafka 版本
#### 版本号
`kafka_2.13-2.8.0`
其中`2.13`是Scala编译器版本，`2.8.0`是Kafka版本号。

`2.8.0`
其中`2`是主版本号， `8`是次版本号， `0`是修订版本号。
注意：`0.10.2.2`的主版本号是`0.10`。
