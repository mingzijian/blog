### 产研线上线下整体质量指标概览   

## 线上指标概况

| 指标名称   | 指标定义                                                     | 数值  | 指标参考                                                     |
| :--------- | :----------------------------------------------------------- | :---- | :----------------------------------------------------------- |
| 线上bug数  | 衡量整体线上质量                                             | 36个  | 79/36=2.19分/个，线上bug平均对应的为normal-major级别，相比于2022年12月（bug数52个，质量缺陷分132分）线上bug数、质量缺陷分下降较明显，单bug的严重程度仍然没有达到normal以下水准 |
| 质量缺陷分 | 衡量产品的线上问题的严重程度计算公式：∑（线上bug**严重程度）* | 79分  |                                                              |
| 缺陷逃逸率 | 衡量产品整体漏测情况（排除故障数量）计算公式：∑（线上bug**严重程度）/（∑（线上bug**严重程度）+∑（线下bug*严重程度）） | 2.54% | 相比于2022年12月（3.1%）缺陷逃逸率略有下降。考虑线上问题具有滞后性，按季度统计，Q1缺陷逃逸率2.54%行业内指标，逃逸缺陷率一般在5%~10%，小于5%算优秀；大于10%需要提高； |

| 指标名称       | 指标定义                          | 数值   | 指标参考                                                     |
| :------------- | :-------------------------------- | :----- | :----------------------------------------------------------- |

| 版本提测打回率 | 衡量开发的交付质量                | 0.56%  | 版本提测打回率可以反应开发交付过程的质量情况，建议低于1%     |
| 低级bug率      | 衡量开发的低级bug情况             | 0.38%  | 低级bug率可以反应开发交付过程的质量情况，建议低于1%          |
| bugreopen率    | 衡量开发的交付质量                | 1.8%   | bugreopen率可以反应开发交付过程的质量情况，建议低于5%        |
